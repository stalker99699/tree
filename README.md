# tree
This is the code that exports a directory and all its contents into 1 txt file. / Это код экспортируюший директорию и всё её содержимое в 1 txt фаил
код работает исключительно на linux
код разделён на 2 файла tree.py и untree.py 

как работает tree.py(упаковшик)

python3 tree.py путь/до/директори/ #и код создаст txt файл директории в кода(где он находится)

как работает untree.py(распаковшик)

python3 untree.py путь/до/файла.txt #и код создаст директорию со всем содержимым на свои места
